# Template: AI Agent App Template

This repository is configured as a GitHub template repository. Use it to bootstrap new AI-agent projects with a consistent structure.

Quick start
- Web: open the repository page and click "Use this template" to create a new repository.
- GitHub CLI: `gh repo create <new-repo-name> --template haroon-mirza/ai-agent-app-template`

Post-create checklist
- Update repository name and `README.md` with the project-specific description.
- Add or rotate secrets (API keys, tokens) in the new repository's Settings → Secrets.
- Review `pyproject.toml` and dependencies.

Notes
- Creating a repository from this template copies files but does not preserve git history.
